ACCOUNTANT ROLE - QUICK START GUIDE
===================================

Login Credentials
- Role: accountant
- Access URL: /public/accountant/accountant_dashboard.php
- Dashboard links to all payroll functions

Main Features
1. Dashboard (/accountant/accountant_dashboard.php)
   - View total employees: 2
   - View monthly payroll: ₹69,500.00
   - View payslips generated count
   - Quick action buttons to all modules

2. Payroll Management (/accountant/payroll_management.php)
   - View all employee salary details
   - Search by employee name
   - Filter by department
   - Summary cards: Total employees, Total payroll, Active accountants, Current month
   - Responsive table with employee info, designation, dept, salary, role, email

3. Financial Reports (/accountant/financial_reports.php)
   Three report types:
   
   a) Payroll Summary
      - Total employees count
      - Total monthly payroll (₹69,500.00)
      - Average salary calculation
      - Min/Max salary range
      - CSV export

   b) By Department
      - Department-wise payroll
      - Employee count per dept
      - Total and average salary per dept
      - CSV export

   c) Employee Details
      - Full employee records with salaries
      - Search by name
      - Filter by department
      - CSV export

4. Generate Payslip (/accountant/generate_payslip.php)
   - Template available for implementation
   - Will support: salary calculation, deductions, bonuses, email delivery

5. Employees (/admin/employees.php)
   - View employee records
   - Access salary information
   - Contact details

Sidebar Navigation
- Dashboard (home page)
- Payroll Management (salary details)
- Financial Reports (analytics and exports)
- Generate Payslip (payslip creation)
- Employees (employee roster)
- Logout (exit system)

Database Access
- Accountants can read:
  * employees table (all fields including salary)
  * departments table (department info)
  * users table (employee roles)
- Accountants cannot modify data (read-only access)

Quick Stats Available
- Total Employees: 2
- Total Monthly Payroll: ₹69,500.00
- Total Departments: 5
- Active Accountants: 1

Export Functionality
- All reports support CSV download
- Filename includes timestamp for tracking
- Format: report_type_YYYYMMDD_HHMMSS.csv
- Integrates with accounting software

Filtering & Search
- Real-time employee name search
- Department dropdown filter
- Combined filtering (search + dept)
- Case-insensitive search

Role Badges
- Shows each employee's system role
- Color-coded for easy identification:
  * Accountant: Blue
  * Director: Dark Blue
  * Administrator: Red
  * Employee: Gray

Common Tasks
1. Check monthly payroll
   Action: Dashboard -> View "Monthly Payroll" stat card

2. Find employee salary
   Action: Payroll Management -> Search by name or filter by dept

3. Generate payroll report
   Action: Financial Reports -> Select "Payroll Summary" -> Export CSV

4. Department-wise analysis
   Action: Financial Reports -> Select "By Department" -> Review breakdown

5. Get employee contact info
   Action: Employees -> View employee records
